This is a landing page project for a beauty brand called "Berrybeauté" built with [Next.js](https://nextjs.org), TypeScript, TailwindCSS, and Motion Frame. Some of its main features are Responsive Design, Product Catalog with Filters, Interactive Hero Banner, Asymmetric Storytelling Timeline.

## Tech Stack

Next.js (Typescipt), Tailwind CSS, Framer Motion

## Features

Responsive Design, Product Catalog with Filters, Interactive Hero Banner, Asymmetric Storytelling Timeline.
